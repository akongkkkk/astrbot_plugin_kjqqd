# 更新日志

## v1.0.3

### ✨ 新增
- 自动签到完成后发送通知到配置的群组
- 添加 `notify_group` 配置项
- 使用 `StarTools.send_message_by_id()` 发送主动消息

### 🐛 修复
- 修复自动签到未执行的问题
- 改用 `asyncio.create_task()` 替代 `cron_manager`
- 添加启动时签到检查机制
- 添加签到日期记录避免重复签到

### 📝 配置方法
在 `config.json` 中设置通知群组：
```json
{
  "notify_group": "群组ID"
}
```
